# Speech
A program for controlling your computer using voice. In Visual Basic 6. 
<br><b>Supports from Windows 95 up to Windows 11 (Windows 95, Windows 98, Windows NT 4.1+, Windows 2000, Windows ME, Windows XP, Windows Vista, Windows 7, Windows 8 and 8.1, Windows 10, Windows 11)</b>
<br><b>Supports Windows Server: Windows Server NT 4.0, Widows Server 2000, Widows Server 2003, Widows Server 2008, Widows Small Business Server 2008, Widows Server 2012, Widows Server 2012 R2, Widows Server 2016, Widows Server 2019, Widows Server 2022, Widows Server 2025
<br><i>Note: Windows 95 and Windows 98 requires Microsoft Speech API, which will be included in installer.</i>

# Principles
* Maximum speed, ease-of-use and efficiency
* Easy installation and portability - no dependencies need to be installed by user
* Overall user convenience and ease of use, yet advanced management for admins and superadmins
* Past, present and future compatibility with Windows

# Why?
Some people do not know how to use their computer using keyboard and mouse to this day. I hope this program will help such people.
